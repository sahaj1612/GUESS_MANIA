# Guess Mania

Guess Mania is a browser-based game project that includes multiple guessing challenges such as word, number, and jumble games with a login-based user flow.

## Tech Stack

- Node.js
- Express
- EJS
- MongoDB
- JWT

## Features

- User signup and login
- Cookie-based authentication
- Word guessing game
- Number guessing game
- Jumbled word game

## Run Locally

```bash
npm install
node route.js
```

Make sure MongoDB is running locally at `mongodb://127.0.0.1:27017/gamecenter`.

Then open `http://localhost:8080`.

## Note

This project is a fun full-stack practice app built around simple game mechanics.
